## Installation

- [Node.js](https://nodejs.org/) 16 or higher
- [npm](https://www.npmjs.com/)

```bash
# Clone this repo.

# Enter the repository.

# Install dependencies.
npm install / yarn install

# Build Package
cd node_modules/hoyoapi
npm install / yarn install
npm run build / yarn build
```
